# AURA Landing Page
Landing page oficial de AURA - Aula Universitaria de Refuerzo Academico.
## Objetivo
Captar interesados en clases virtuales de Fisica y Matematica para escolares,
preuniversitarios y universitarios.
## Estructura
- index.html: pagina principal.
- assets/: carpeta de imagenes y recursos.
## Embudo
Google Ads -> Landing Page -> WhatsApp -> Registro manual del alumno.
## Medicion
La landing esta preparada para medir clics al boton de WhatsApp mediante Google
Tag Manager y Google Ads.
## Estado del proyecto
Primera version en desarrollo.
